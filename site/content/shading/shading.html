"A Shader is a user-defined program designed to run on some stage of a graphics processor. Shaders provide the code for certain programmable stages of the rendering pipeline. They can also be used in a slightly more limited form for general, on-GPU computation." - Khronos.org

An introduction to shading

Shaders are programms written in GLSL (GL Shading Language) which is a dialect of the C programming language. They are used to control the rendering process of 3D sceneries and are applied in multiple steps. These are the shaders available for application during the rendering process:

  • Vertex Shader (required)
  • Tessellation Shader
  • Geometry Shader
  • Fragment Shader (required)
  • Compute Shader

The shaders listed above are ordered in the way they are to be applied during the rendering process. The following page however will only elaborate on the functionality of the Vertex and Fragment Shader, since the others are optional.
